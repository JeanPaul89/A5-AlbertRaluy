CSS

CSS, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

Partes Principales de CSS

Selectores:

Mediante los cuales podemos especificar qué elementos de la página nos estamos refiriendo.

Atributos:

Sirven para definir qué cosas queremos estilizar sobre los selectores indicados

Valores:

Indican qué estilo se debe aplicar a cada atributo sobre cada selector.
